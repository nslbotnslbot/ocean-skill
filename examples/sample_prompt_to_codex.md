# Sample prompt to Codex

Use the OCEAN skill to evaluate this project/manuscript.

Please output in Chinese.
Use the standard OCEAN output format unless I explicitly ask for quick mode or deep mode.

Focus on:

1. What scientific problem it actually solves.
2. Whether this is a system/tool, method, resource, application validation, or scientific discovery.
3. Whether the evidence supports the main claims.
4. Whether there is data leakage, weak validation, benchmark unfairness, or causal overstatement.
5. Whether database/text-mining evidence is being overinterpreted as mechanism.
6. What analyses must be added.
7. How I should participate: light, medium, deep, or authorship-level.
8. Journal positioning: stretch, realistic, backup.
9. Scores from 0 to 10.

Do not just summarize. Be critical and concrete.
