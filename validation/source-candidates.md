## Use Notes

- Do not paste large source passages into the skill repository.
- For each eval run, record the exact public URL and the inspected evidence boundary.
- If only metadata or abstract is inspected, label the run as a metadata-only or abstract-only test.
- Do not infer sample sizes, validation cohorts, reviewer recommendations, or author claims from titles alone.
- Prefer adding new candidates only when they cover a missing eval behavior.
