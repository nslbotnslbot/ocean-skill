# OCEAN Differentiation Note

OCEAN is intentionally scoped as an external biomedical claim-evidence navigation workflow, not a general AI-science execution framework.

## Positive Direction

OCEAN should be described as:

- a source-boundary and evidence-navigation skill;
- a claim-risk triage layer for medical and biological research;
- a reviewer-style audit workflow for manuscripts, preprints, databases, knowledge graphs, clinical prediction models, and biomedical AI studies;
- a validation-planning and collaboration-boundary assistant.

Its main public identity is: **navigate sources, validate claim boundaries, and decide what evidence is still missing.**

## Differentiation Guardrails

OCEAN should avoid product framing centered on:

- execution layers;
- execution-drafted evidence ledgers;
- guidance-reviewed release gates;
- manuscript-facing release authority;
- endpoint-calibration systems;
- outcome-spectrum labels;
- source-proof-card publishing;
- claim-after / paired-non-claim ledgers.

These are treated as adjacent research-agent ideas rather than OCEAN's identity.

## Module-level Distinction

| OCEAN module | Differentiated role |
|---|---|
| Sounding | Builds source packets and negative-space maps from traceable literature/review/database inputs |
| Current | Reads direction flow only within explicit search/source boundaries |
| Reef | Organizes external resource, KG, database, cohort, and benchmark provenance |
| Iceberg | Audits claim support and hidden overclaim risk |
| Anchor | Converts gaps into validation, leakage, replication, and benchmark checks |
| Compass | Turns bounded evidence into research planning and strategy |
| Harbor | Preserves review reports and contribution-boundary notes, not release authority |

## Public Wording Rule

For GitHub, docs, evals, and examples, prefer:

> OCEAN is a biomedical claim-evidence navigation workflow.

Avoid:

> OCEAN is an execution-ledger, release-gated, endpoint-calibration system.

## Boundary

This note is a positioning and similarity-avoidance guardrail. It is not a scientific claim and does not evaluate any external system.
