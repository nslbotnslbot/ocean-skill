## Public-Use Rules

- Do not copy large source passages into the repository.
- Do not represent synthetic adversarial prompts as author claims.
- If only metadata or abstract evidence was inspected, say so clearly.
- Do not infer sample sizes, clinical validation, peer-review outcomes, or author conclusions from titles alone.
